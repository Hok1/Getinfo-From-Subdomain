>用途：

>用于收集子域名后的信息收集。收集信息后输出到文档。

>用法：
```
python3 getinfo_subdomain.py -o 输出文件名
```

>eg:
```
python3 getinfo_subdomain.py -o result.txt
```

>more details:
```
python3 getinfo_subdomain.py
```
